# 质量与长期体验信号

以下 17 项均为 `layer=analysis_only`、`detectability=low`。它们通常依赖连续中段或全书证据，不是快扫首尾样本能可靠确认的用户选项。用户可以自然语言声明讨厌某类写法，但快速定位仍必须写“当前覆盖不足以判断”。

- `quality.repetitive_face_slap`｜打脸流程重复：反复使用“受辱—亮实力或身份—众人震惊”的同构循环。
- `quality.excessive_showboating`｜强行装逼过多：大量无关场景只为展示主角优越，且不推进关系或目标。
- `quality.villain_logic_sacrificed`｜反派为送经验而降智：反派行为违背已知能力、利益和信息，只为让主角获胜。
- `quality.supporting_cast_instrumentalized`｜配角工具化：配角长期缺乏自身目标，只负责衬托、送资源或推动主角。
- `quality.female_characters_instrumentalized`｜女性角色工具化：女性角色主要负责倒贴、受害或制造麻烦，缺少独立选择。
- `quality.power_scaling_breakdown`｜战力体系崩坏：同类能力和境界前后失去一致标准，结果随剧情任意变化。
- `quality.realm_reskin_loop`｜境界套娃：升级只重复换名，规则、挑战和阅读体验没有实质变化。
- `quality.plot_padding`｜注水严重：同一事件被重复描写，章节信息增量长期过低。
- `quality.main_plot_stagnation`｜主线推进慢：大量章节不改变核心目标、局势或角色选择。
- `quality.daily_life_overweight`｜日常戏份过多：日常内容长期压过作品承诺的主要剧情。
- `quality.authorial_preaching`｜作者说教过多：角色或旁白频繁脱离具体情境输出价值判断。
- `quality.childish_wish_fulfillment`｜爽点过度幼稚：世界和角色系统性失真，只为配合主角获胜或受追捧。
- `quality.cliched_melodrama`｜狗血套路堆叠：失忆、误会、错过等陈旧桥段反复承担剧情推进。
- `quality.prose_flat_or_incoherent`｜文笔或表达较差：长期流水账、逻辑断裂或严重重复；不能把个人文风偏好当证据。
- `quality.rushed_ending`｜草率收尾：大量已建立的问题、关系和伏笔未经处理便突然结束。
- `quality.character_consistency_break`｜人设前后断裂：关键选择与既有价值观、能力和关系明显冲突，文本又没有展示充分变化过程。
- `quality.disguise_backfires`｜扮猪反成受气：主角隐藏实力却长期承受无意义羞辱，所得收益不足以解释该选择。

## 快扫禁区

只读开头与物理末尾时，禁止用这些标签断言“全书水、反派都蠢、女主没有人格、作品烂尾、配角全是工具人”。末尾番外是结局样本，不是“只写了十章就提前剧透”。若用户的已确认偏好依赖这些信号，加入决定性未知并建议连续扫描。
